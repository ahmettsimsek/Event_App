## 1. expo Paketini Yükle
```
  npm install expo
```
veya
```
 yarn add expo
```

## 2. Diğer Bağımlılıkları Yükle
Projede eksik olan diğer bağımlılıkları da yüklemek için:
```
   npm install
```
veya
```
yarn install
```

## 3. Projeyi başlat
```
npx expo start
```
